body 요소는 문서의 본문을 담는 요소로 아래와 같이 [html 요소](/html/html.html) 안에, [head 요소](/html/head.html) 다음에 적습니다.

{% highlight html %}
<!DOCTYPE html>
<html>
  <head>
    ...생략...
  </head>
  <body>
    ...이곳에 본문 내용을 적습니다...
  </body>
</html>
{% endhighlight %}

HTML 문서의 전체적인 구조에 대한 설명은 [HTML 문서 기본 구조](/docs/HTML_boilerplate.html)를 참고하세요.
